== Installation ==

Note : This plugin works with Cornerstone Page Builder and Pro theme. Make sure you have <a href="https://codecanyon.net/item/cornerstone-the-wordpress-page-builder/15518868?ref=Codetic" rel="nofollow">Cornerstone</a> installed. It's compatible with <a href="https://theme.co/pro/">Pro</a> as well.


1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You can type "EACS" on your element tabs within Cornerstone editor and all the avialable elements will appear.

== Frequently Asked Questions ==

= Can I use the plugin without Cornerstone Page Builder? =

No. You cannot use without Cornerstone but if you have Pro then Cornerstone is not required.

= Does it work with "Pro" theme? =

Absolutely! It's compatible with Pro as well.

= What if I update to Premium version? =

Your existing elements/content will work with premium version. So you won't lose your developed contents.
